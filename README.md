# OiGO — Visual AAC Communication App

A visual AAC (Augmentative and Alternative Communication) app built for children 
with autism who understand and respond better to visual information than verbal questions.

**"Oigo" means "I hear" in Spanish.**

## Features
- 🎤 Voice input — speak your question and answer options
- 🔊 Text-to-speech — reads questions and answers aloud  
- 📷 Photo upload — use real photos instead of emoji for answer cards
- 💾 Saved sets — save and reuse your favorite question sets
- 📋 Response log — track answers for therapy notes
- 🔲 Big card mode — extra large icons for easier tapping
- 📵 Works offline — full PWA, installable to home screen
- 🎨 Blue & green design — calm, clear, easy to read

## Deploy to GitHub Pages
1. Upload all files to a GitHub repository
2. Go to **Settings → Pages → Source → main branch**
3. Your app will be live at `https://yourusername.github.io/repo-name`

## Install on iPhone (Safari)
1. Open your GitHub Pages URL in **Safari**
2. Tap the **Share button** (box with arrow)
3. Tap **"Add to Home Screen"**
4. Tap **Add** — OiGO installs as a real app with its own icon

## Files
- `index.html` — the full app
- `manifest.json` — PWA manifest
- `sw.js` — service worker for offline support
- `icon-192.png` / `icon-512.png` — OiGO app icons
